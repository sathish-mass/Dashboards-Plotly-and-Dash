## Getting Started

### Prerequisites

This is the list things you need to install them.
* pandas
  ```sh
  pip3 install pandas
  ```

### Installation

1. Install plotly
   ```sh
   pip3 install httpx==0.20 dash plotly
   ```
2. Install packages
   ```sh
   pip3 install packaging
   ```
3. Install Dash
   ```sh
   pip3 install pandas dash
   ```
